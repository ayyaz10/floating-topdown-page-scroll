# Floating Topdown Page Scroll

A small Chrome extension that enables floating top-down page scrolling behavior.

## Files

- `manifest.json` - Chrome extension metadata
- `content.js` - Main content script that handles scrolling behavior

## Installation

1. Open `chrome://extensions/` in Chrome.
2. Enable **Developer mode**.
3. Click **Load unpacked**.
4. Select this extension folder.

## Development

- Edit `content.js` and `manifest.json`.
- Reload the extension on the Chrome extensions page after changes.

## Notes

- This repo is intended for a Chrome extension and does not require any build tools by default.
- Add `node_modules/` or build folders to `.gitignore` if you later add dependencies or tooling.
